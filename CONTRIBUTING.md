# Contributing to Agent RedTeam

We welcome contributions! Please follow these guidelines:

1. **Fork** the repository.
2. **Feature Branch**: Create a branch for your feature/bugfix.
3. **Commit**: Follow [Conventional Commits](https://www.conventionalcommits.org/).
4. **Test**: Run `cargo test --all-targets --all-features` before submitting.
5. **PR**: Open a Pull Request with a clear description of the security impact.
